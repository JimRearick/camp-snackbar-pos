# Camp Snack Bar POS System

A complete, local-network-only point-of-sale system for summer camps.

## Quick Start

```bash
# Build and run
docker-compose build
docker-compose up -d

# Access from tablets
http://[SERVER-IP]:5000
```

**Default admin password:** `camp2024` (CHANGE IMMEDIATELY)

## Features

- ✅ Multi-tablet real-time sync
- ✅ Touch-optimized interface
- ✅ Account management (family/individual)
- ✅ Product categories
- ✅ Automated backups
- ✅ Comprehensive reporting
